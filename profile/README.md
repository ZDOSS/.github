## About

ZDOSS is a small organization building focused open-source software. We work across game development tooling, creative automation, and public data ‚ÄĒ with an emphasis on clear interfaces, practical scope, and code that fits real workflows.

There is no single product mandate. Projects ship when they solve a problem well.
